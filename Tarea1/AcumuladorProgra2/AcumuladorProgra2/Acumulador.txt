Procedimiento para hacer la Tpg1:
================================

- http://www.di-mare.com/adolfo/cursos/2009-1/p2-ta-1.htm

Jalar el archivo de la Tpg1:
- http://www.di-mare.com/pub/Pg2/Acumulador.zip
- hago una carpeta "Acumulador"
- Adentro copio lo que está en Acumulador.zip

- Edito el archivo Acumulador.h
  - Uso el editor de Code::Blocks
     - Marco el contenido completo de Acumulador.h
	 - Uso AStyle para darle formato
	   ==> Plugins ==> Source code Formater (AStyle)
  - Uso Notepad++
  - Le corrijo el espaciado y la indentación
    - Modo de Búsqueda
  	- [x] Extendido (\n, \r, \t, ...)
  	  - Buscar [;] Remplazar con [;\n]
  	  - Buscar [{] Remplazar con [{\n]
  	  - Buscar [}] Remplazar con [}\n]
  - TestFX ==> TestFX Edit ==> Reindent C++ code

- Para compilar el programa uso el proyecto .cbp
  - [[ También puedo compilar usando el proyecto .dev ]]
- Compilador de C++ ==> GNU C++  (NO VS ni Cywin)

- Jalo el archivo
  - http://www.di-mare.com/adolfo/p/BUnit/BUnit.h
- Lo arreglo en la carpeta

- Se compila con Build
- Se ejecuta con F7

Ahí adentro está un ejemplo de cómo hacerlo:

00029    /** Inicializa en cero el acumulador.
00030       \dontinclude test_Acumulador.cpp
00031       \skipline    test::borre()
00032       \until       }}
00033       \see        test_Acumulador::test_borre()
00034    */
00035    Acumulador() {  borre(); }

Los comentarios Doxygen están dentro de /** */ o comienzan con ///

Si les da calambre la palabro "Doxygen", pueden encontrar todo lo que buscan en la red:
- http://search.yahoo.com/search?n=100&p=Doxygen

La forma en que yo genero el archivo Doxygen es usando el siguiente comando desde la pantalla negra:

C:\DIR\SubDir> doxygen Acumulador.dxg

[[[ Por supuesto, esto funciona si los fuentes C++ y el archivo ]]]
[[[ Acumulador.dxg están en la carpeta "C:\DIR\SubDir"         ]]]

El archivo "Acumulador.dxg" es el que les di en clase, el que también está en el enunciado de la Tpg1.

Ustedes pueden usar el DoxyWizard si así lo desean.

Recuerden: Basta que modifiquen Acumulador.h incluyéndole la documentación Doxygen para que terminen la Tpg1. La especificación de cada método está en la carpeta html-Adolfo del .zip que les entregué en clase.
